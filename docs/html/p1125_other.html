Other/Misc¶

Sending Logs¶



  1. Logs are used to help diagnose problems.

  2. Access sending logs via Menu->Send Logs.


















Note

P1125 logs contain no user measurement data. Only the state of the P1125 is reported in the logs. The P1125 serial number and software version are recorded in the logs.

Note

P1125 logs are only sent by User action.

Previewing Logs¶

You may ssh into the P1125 to preview the logs that will be sent. All of the files in the log folder will be sent in a zip file.

$ ssh pi@p1125.####.local
pi@p1125-####:~ $ ls -al log

Note

The P1125 pi user password is pi.

Note

The zip file is encrypted when it is sent to the Sistemi log server.

Shutting Down¶

The P1125 runs on a Raspberry Pi 4B running standard Raspberry Pi OS on an SD card.

It is best practice to shutdown the OS via Main->Shutdown.
