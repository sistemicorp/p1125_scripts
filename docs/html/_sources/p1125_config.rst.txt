Configuration
#############

Setting Presets
---------------

.. index:: presets, preferences

.. figure:: _static/p1125_config_presets_01.png
    :figwidth: 40%
    :align: right


Some controls of the P1125 have `preset` values that can be set to an alternate default value.
Setting these `presets` may improve your efficiency with the P1125.

1. Access the `Preferences` page via the Menu->Preferences.

2. The available `presets` will be shown with controls to adjust them.
3. Press the `SUBMIT` button to store the results in the `app_config.json` file.
4. To go back to the main page via `Menu->Main`.
